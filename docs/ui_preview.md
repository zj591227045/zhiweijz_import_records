# 导入交易记录网页 - 界面预览

## 整体设计风格

### 设计原则
- **简洁明了**: 清晰的视觉层次，减少认知负担
- **步骤引导**: 明确的流程指示，降低操作难度
- **即时反馈**: 实时的状态更新和错误提示
- **响应式设计**: 适配多种设备屏幕

### 色彩规范
- **主色调**: #409EFF (Element Plus 蓝色)
- **成功色**: #67C23A
- **警告色**: #E6A23C  
- **错误色**: #F56C6C
- **中性色**: #606266, #909399, #C0C4CC

## 页面流程设计

### 导航流程
```
首页 → 服务器配置 → 用户登录 → 账本选择 → 文件上传 → 分类映射 → 数据预览 → 导入确认 → 导入报告
```
